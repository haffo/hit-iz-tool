Description
The initial Laboratory Test Compendium is delivered electronically from the LIS to the EHR-S. The EHR will integrate the eDOS into its test directory and use it to allow placement of orders electronically. The initial laboratory test compendium is composed of up to four messages.
Comments
This is the initial "smoke testing" to reveal simple failures severe enough to preclude further testing.
PreCondition
No Pre-Condition.
PostCondition
No Post-Condition.
TestObjectives
  • Demonstrate capability to support minimally populated messages (single occurrences of all required ("R") elements).
Notes to Testers
"Smoke test" Minimal test for required data elements. This is using replace messages, so all data in these messages will replace information from a previous eDOS upload

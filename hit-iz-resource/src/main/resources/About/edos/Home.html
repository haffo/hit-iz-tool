Supporting

The NIST Labs Test Suite supports the testing of HL7 v2.5.1 messages in support of the Labs Community.

  • HL7 Version 2.5.1 Implementation Guide: S&I Framework Laboratory Test Compendium Framework R2, DSTU Release 2 - US Realm
  • HL7 Version 2.5.1 Implementation Guide: S&I Framework Laboratory Orders (LOI) from EHR, Release 1, DSTU Release 2 - US Realm
  • HL7 Version 2.5.1 Implementation Guide: S&I Framework Lab Results Interface, Release 1, DSTU Release 2 - US Realm
Have a Question?

Three Google Groups have been established for discussion/questions about the Implementation Guides, testing tool, and testing issues. No membership is required. A Google account is required for posting. When asking questions, please specify the edition of the tool and version of the standard to which the questions apply.

  • eDOS Site: https://groups.google.com/d/forum/hl7v2-lab-compendium-testing
  • eDOS Email: hl7v2-lab-compendium-testing@googlegroups.com
  • LOI Site: https://groups.google.com/d/forum/hl7v2-lab-orders-interface-testing
  • LOI Email: hl7v2-lab-orders-interface-testing@googlegroups.com
  • LRI R2 Site: https://groups.google.com/d/forum/hl7v2-lab-testing
  • LRI R2 Email: hl7v2-lab-testing@googlegroups.com
